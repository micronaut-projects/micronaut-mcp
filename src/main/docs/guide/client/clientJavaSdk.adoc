To use MCP Clients powered by the https://github.com/modelcontextprotocol/java-sdk[MCP Java SDK], use the following dependency:

dependency:micronaut-mcp-client-java-sdk[groupId=io.micronaut.mcp]

You can configure multiple clients via configuration:

include::{includedir}configurationProperties/io.micronaut.mcp.client.javasdk.McpClientHttpConfigurationProperties.adoc[]

Additionally, if you are building an <<serverJavaSdk, MCP Server>>, you can test it by injecting a bean of type `io.modelcontextprotocol.client.McpSyncClient` or `io.modelcontextprotocol.client.McpAsyncClient` in your tests. The MCP Client pointing to your MCP Server has a https://docs.micronaut.io/latest/guide/#qualifiers[name qualifier] with value `embeddedServer`.
