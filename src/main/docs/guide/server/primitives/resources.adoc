____
Resources: Structured data or content that provides additional context to the model
____

The preferred way to define resources is to annotate a method with api:mcp.annotations.Resource[] in a `@Singleton` bean. See <<resourcesAnnotations, Resources with Annotations>>.

Alternatively, you define resources registering beans (typically in a https://docs.micronaut.io/latest/guide/#factories[bean factory]).
