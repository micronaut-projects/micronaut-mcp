NOTE: Micronaut MCP binds parameters with type api:mcp.server.context.MicronautMcpTransportContext[] or `io.modelcontextprotocol.spec.McpSchema.GetPromptRequest` automatically.

Alternatively, you define prompts registering beans (typically in a https://docs.micronaut.io/latest/guide/#factories[bean factory]) using the low-level MCP SDK API.

NOTE: The following example assumes you set `SYNC` as your server type.
