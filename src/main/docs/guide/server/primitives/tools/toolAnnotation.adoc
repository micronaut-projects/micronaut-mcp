By default, the method name is used as the tool name.
You can override this by setting the `name` attribute of the `@Tool` annotation.

WARNING: Defining a tool via an annotation currently does not work with https://docs.micronaut.io/latest/guide/#eagerInit[eager initialization of singletons].
